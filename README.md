Mini Projet — Text Mining
=========================

But
----
Comparer les représentations classiques (TF-IDF) et sémantiques (Word2Vec, GloVe, fastText) pour la classification spam/ham. Utiliser un RNN (LSTM) pour les embeddings.

Fichiers fournis
----------------
- `mini_projet_text_mining.ipynb` : Notebook principal prêt à exécuter.
- `requirements.txt` : dépendances Python.

Comment exécuter
----------------
1. Créez un environnement Python (recommandé `venv` ou `conda`).
2. Installez les dépendances :

```bash
pip install -r requirements.txt
```

3. Ouvrir `spamham.ipynb` dans JupyterLab / Jupyter Notebook et exécuter les cellules dans l'ordre.
